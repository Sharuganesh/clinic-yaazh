## 5-Minute Deployment

### 1️⃣ Create GitHub Repository (2 minutes)

```bash
cd C:\Users\sharu\Desktop\Yaaz_clinic-main

# Initialize git
git init
git add .
git commit -m "Yaaz Clinic - Ready for Vercel"

# Create repo at GitHub.com and then:
git remote add origin https://github.com/YOUR_USERNAME/yaaz-clinic.git
git branch -M main
git push -u origin main
```

### 2️⃣ Deploy to Vercel (3 minutes)

**Via Dashboard (Easiest):**
1. Go to https://vercel.com
2. Sign in with GitHub
3. Click "New Project"
4. Select your `yaaz-clinic` repository
5. Click "Deploy"
6. **Done!** Your site is live ✨
